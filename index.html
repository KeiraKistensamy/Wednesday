There are different programming languages that involve web development

Javascript

Javascript is responsible for creating web pages. It allows us to add dynamic behaviour to the webpage and add special effects to the webpage

HTML

The HTML provides structue to a website. This includes text, images, videos, multimedia etc

CSS

CSS is used to style and layout web pages. Its responsible for text style, positioning, color etc. It also controls how a website's style shifts between desktop and mobile versions.
